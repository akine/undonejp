制作に関する規約

以下は、合同会社Undoneの制作業務に関する規約全文です。

■共通規約 サービス提供条件および重要事項

本規約は、合同会社Undone（以下「弊社」）が提供するすべてのサービスに共通して適用される条件および重要事項を定めるものです。

第1条（契約の成立について）
1. 弊社が提示した見積書に対し、お客様より発注意思（書面、メール、チャットツール等を含む）をいただいた時点で契約成立といたします。  
2. 個別契約書または見積書に別途定めがある場合は、当該内容を優先いたします。

第2条（著作権および利用権について）
1. 弊社が制作した成果物の著作権（著作権法第27条および第28条を含む）は、特段の定めがない限り弊社に帰属します。  
個別契約書または見積書に別途定めがある場合は、当該内容を優先いたします。
2. お客様には、契約時に合意した媒体・用途・期間の範囲内での利用権を許諾いたします。  
3. 媒体追加、用途変更、広告利用、二次利用、再編集等を行う場合は、別途契約または追加費用が発生する場合がございます。  
4. 弊社は制作実績として、成果物をWebサイト、SNS、営業資料、ポートフォリオ等へ掲載できるものとします。非公開をご希望の場合は事前にご相談ください。

第3条（支払および解除条件について）
1. お支払いは原則として請求書発行日より60日以内といたします。  
個別契約書または見積書、請求書等に別途定めがある場合は、当該内容を優先いたします。
2. 支払期限を超過した場合、年14.6％（法令の範囲内）の遅延損害金をご請求する場合がございます。
3. 支払遅延または重大な本規約違反がある場合、弊社は納品データの引渡し停止、編集データ・素材・マスター等の提供停止、ならびに利用許諾の停止（ライセンス停止）を行うことができます。
4. 当該事由が解消された場合、弊社は合理的な範囲で停止を解除します。
5. お客様が本規約に違反し、相当期間を定めて是正を求めても改善がない場合、弊社は契約を解除できるものとします。
6. お客様が反社会的勢力に該当する場合、または関与していることが判明した場合、弊社は催告なく契約を解除できます。

第4条（お客様提供素材について）
1. お客様より提供された映像、画像、音源、資料等に関する著作権・肖像権・使用許諾等の権利処理は、お客様の責任にて行うものとします。  
2. 提供素材に起因する第三者との紛争について、弊社は責任を負いかねます。

第5条（再委託について）
1. 弊社は業務遂行に必要な範囲において、業務の全部または一部を第三者へ再委託できるものとします。

第6条（秘密保持について）
1. 弊社およびお客様は、業務上知り得た非公開情報を第三者へ開示または漏洩しないものとします。  
2. 法令に基づく開示要請がある場合を除き、本条の義務は契約終了後も継続するものとします。

第7条（免責事項について）
1. 天災地変、停電、通信障害、配信プラットフォーム障害、機材故障その他弊社の合理的支配を超える事由により発生した損害について、弊社は責任を負いかねます。  
2. 前項の事由により業務遂行が困難となった場合、スケジュール変更または業務内容を調整させていただく場合がございます。
3. 配信業務は、善良なる管理者の注意義務に基づき最善の運用を行うものとし、視聴数、通信品質、配信結果、収益等の成果を保証するものではありません。
4. 成果物の内容、出演者の発言、企画内容等に起因する第三者からの請求、権利侵害、名誉毀損等については、お客様の指示または提供内容に基づく場合、弊社は責任を負いません。
5. 弊社は、第8条にかかわらず、特別損害・間接損害・逸失利益（機会損失、売上減少等を含む）について責任を負いません。

第8条（損害賠償の上限について）
1. 弊社の責任により損害が発生した場合の賠償責任は、当該案件の契約金額を上限とします。

第9条（規約の変更について）
1. 弊社は必要に応じて本規約を変更できるものとします。  
2. 変更後の規約は、弊社Webサイトへ掲載した時点で効力を生じるものとします。

第10条（準拠法および管轄について）
1. 本規約は日本法に準拠します。  
2. 本規約または本契約に関連して生じた紛争については、東京地方裁判所を第一審の専属管轄裁判所とします。


■動画制作・編集 制作条件および重要事項

本規約は、動画制作（企画・撮影・編集）および動画編集業務に関する条件および重要事項を定めるものです

第1条（制作・編集体制および工数について）
1. 本見積の作業工数（人日）は、一般的な1日8時間の労働時間概念および広告・映像制作業界の制作ガイドラインを基準として算出しております。  
2. 立会い編集やオンラインでの確認作業が上記基準（8h）を大幅に超える場合、または深夜（22:00〜翌5:00）や休日（土日祝）の稼働が必要となった場合は、別途割増料金をご相談させていただく場合がございます。
3. お客様からの素材提供、確認、承認の遅延により納期へ影響が生じた場合、当初納期は自動的に延長されるものとします。
4. 素材支給または確認遅延によりスケジュール変更が生じた場合、追加費用が発生する場合がございます。

第2条（修正および検収について）
1. 本見積には、初号納品後の修正作業を原則「2回まで」含んでおります。  
2. 3回目以降の修正、または検収（内容承認）完了後の修正、および当初の企画・仕様を大幅に変更する修正については、実作業時間（h）に応じた追加費用を別途申し受けます。  
3. プレビュー提出後、7日以内（見積書等で別途定める場合を除く）に修正指示または検収結果の連絡がない場合、当該成果物は検収完了したものとみなします。

第3条（メディア管理およびデータ保管について）
1. 本業務で制作した映像・音声素材および編集プロジェクトデータは、原則として保存いたしません。  
2. 初号原版（マスター）の納品・検収完了をもって、弊社の保管責任および管理責任は終了するものとします。  
3. 納品後の一定期間のデータ保持をご希望の場合は、見積明細の「保管オプション」を事前にご指定ください。  
4. 保管期間満了後は、事前の通知なくデータを自動廃棄、または実費にて返却させていただきます。  
5. 1年を超える長期保管（延長）をご希望の場合は、別途「保管延長費」および「保管延長関連人件費」を申し受けます。  
6. 万全を期して管理いたしますが、不測の事態（メディアの経年劣化、ストレージの故障やデータ破損等）により発生した損害について、弊社の故意または重大な過失によらない限り、責任を負いかねます。

第4条（プロジェクトデータの提供について）
1. 本業務で作成した編集プロジェクトデータ（Premiere Proプロジェクトファイル等）は、弊社の所有する「中間生成物」であり、本契約の納品物には含まれません。  
2. プロジェクトデータの譲渡または提供をご希望の場合は、別途「素材譲渡費」および「データ整理手数料」を申し受けます。  
3. 素材譲渡にあたっては、使用しているフォント、プラグイン、ライブラリー音源等のライセンスおよび使用権利が、お客様の環境に継承されない場合がございます。  
4. データ譲渡後の改変、二次利用、およびお客様側でのデータ取り扱い・保管に起因するトラブルについて、弊社は一切の責任を負いかねますのであらかじめご了承ください。

第5条（納品形式およびエンコードについて）
1. 本見積に含まれる納品形式は、事前に合意した指定形式1種類を基準とします。  
2. 解像度変更、媒体別仕様（Web、SNS、広告媒体、放送用途等）への変換、コーデック変更、容量制限対応、縦横比変更等の書き出し作業（以下「エンコード作業」）は、編集作業とは別工程として取り扱います。  
3. 複数形式での納品、用途別データ作成、追加書き出しが必要となる場合は、見積明細記載の「エンコード費」を適用いたします。  
4. 本見積における「1点」とは、納品形式・解像度・用途ごとの書き出しデータ1ファイルを指します。  
5. エンコード後の軽微な仕様変更（容量調整、ビットレート変更、形式変更等）は、新規エンコード作業として取り扱う場合がございます。

第6条（キャンセルポリシー）
制作・編集業務においては、作業予定日の確保や外注先（スタジオ、ナレーター等）のブッキングを伴うため、制作業界における一般的な慣行を参考として以下の規定を適用いたします。

(1) 編集・仕上げ等の作業日が確定している場合  
・作業日より3日前まで：お見積金額の50%  
・作業日より2日前まで：お見積金額の70%  
・作業日より前日〜当日：お見積金額の100%  

※上記キャンセル料が、既に発生している実費（外部スタジオ、出演者等のキャンセル料実費）を下回る場合は、当該実費を全額ご請求させていただきます。

(2) 正式発注後、企画が中止または延期となった場合  
・それまでにかかった実稼働分の費用（企画構成、素材整理、制作準備等）を100%ご請求いたします。  
・上記に加え、制作管理手数料（プロダクションフィー）として、お見積金額（値引き前）の10%を事務手数料として申し受けます。

■イベント配信規約 配信業務条件および重要事項

本規約は、イベント配信業務に関する条件および重要事項を定めるものです。

第1条（人件費について）
1. 本見積における稼働時間は、設営開始から撤収完了までの拘束時間を基準とします。予定拘束時間を超過した場合、追加人件費および機材費が発生する場合があります。
2. 当日の進行変更、追加演出、回線追加、配信先追加等により作業が増加する場合、弊社の判断により追加費用が発生する場合があります。

第2条（ネットワーク環境について）
1. 配信には原則として会場ネットワークを使用いたします。  
2. 安定配信のため、配信専用ネットワークのご用意を推奨しております。  
3. 推奨回線速度は上下50Mbps以上とします。

第3条（通信品質および免責について）
1. 配信プラットフォーム、通信回線、ISP状況等に起因する配信中断・品質低下について弊社は責任を負いかねます。  
2. 来場者による回線使用増加等により帯域が圧迫された場合も同様とします。

第4条（会場設備および弊社持込機材について）
1. 安定した電源供給、機材設置スペース、必要な入館許可はお客様側にてご準備いただくものとします。  
2. 会場設備に起因するトラブルについて弊社は責任を負いかねます。
3. お客様または来場者、会場関係者の過失により機材の破損・紛失が発生した場合、修理費または再取得費用をご請求する場合があります。

第5条（録画およびデータ管理について）
1. 配信記録データの保管期間は別途定めるものとします。  
2. 保管期間経過後は予告なく削除される場合がございます。
3. バックアップ録画は安全性向上を目的としたものであり、録画データの完全性または取得成功を保証するものではありません。
4. アーカイブ公開、再配信、切り抜き、広告利用等を行う場合の権利処理（出演者・音楽・映像・会場）はお客様の責任で行うものとします。

第6条（キャンセルポリシー）
1. 開催日が確定している場合
開催日より 14日前～11日前まで: お見積金額の30%
開催日より 10日前～6日前まで: お見積金額の50%
開催日より 5日前～3日前まで: お見積金額の80%
開催日より 2日前～当日まで: お見積金額の100%
上記キャンセル料率に基づく金額が、弊社が既に手配・契約済みの機材レンタル費用、外部スタッフ費用、会場費等に発生するキャンセル料・違約金の実費を下回る場合、当該実費（実際の発生費用）をご請求させていただきます。

2. 開催日が確定していない場合
正式発注後、本企画中止または延期の場合、以下の費用が発生いたします。
発生した実稼働分の費用（交通費・宿泊費、手配済み機材費等を含む）を100%ご請求いたします。
上記に加え、全体の制作費（値引きを除くお見積もり金額）の10%をキャンセル事務手数料（プロダクションフィー）としてご請求いたします。
【重要事項】
弊社がイベント会場や撮影スタジオなどを代理で契約している場合、当該施設やスタッフのキャンセルポリシーが別途適用され、その費用はお客様ご負担となります。
